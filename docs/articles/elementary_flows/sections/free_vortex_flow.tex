\section{Free Vortex Flow}
\label{sec:free-vortex}

A free vortex represents purely rotational motion around a point.
The circulation strength is denoted by $\Gamma$.

\bigskip

The velocity field is

\[
V_r = 0,
\qquad
V_\theta
=
\frac{\Gamma}{2\pi r}
\]

\subsection{Velocity Potential}
\label{subsec:free-vortex-flow-velocity-potential}

Using the polar coordinate relation

\[
V_\theta
=
\frac{1}{r}
\frac{\partial\phi}{\partial\theta}
\]

integration gives

\[
\boxed{
\phi
=
\frac{\Gamma}{2\pi}\theta
}
\]

\subsection{Stream Function}
\label{subsec:free-vortex-flow-stream-function}

The stream function becomes

\[
\boxed{
\psi
=
-\frac{\Gamma}{2\pi}\ln r
}
\]
