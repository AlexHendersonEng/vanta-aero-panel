\section{Superposition Principle}
\label{sec:superposition-principle}

Since both the velocity potential and stream function satisfy Laplace's
equation, solutions may be added together.

\bigskip

If

\[
\phi_1,\phi_2,\ldots,\phi_n
\]

are valid velocity potentials, then

\[
\phi
=
\sum_{i=1}^{n}\phi_i
\]

is also a valid solution.

\bigskip

Similarly,

\[
\psi
=
\sum_{i=1}^{n}\psi_i.
\]

This principle allows complex flow fields to be generated from simple
elementary flows.
