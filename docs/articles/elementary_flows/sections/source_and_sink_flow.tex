\section{Source and Sink Flow}
\label{sec:source-flow}

A source is a point from which fluid radiates uniformly in all directions
and a sink is the opposite.
Let the source or sink strength be $Q$ (volume flow rate per unit depth).
Using polar coordinates $(r,\theta)$, the velocity field is

\[
V_r = \frac{Q}{2\pi r},
\qquad
V_\theta = 0
\]

\subsection{Velocity Potential}
\label{subsec:source-and-sink-flow-velocity-potential}

Since

\[
V_r = \frac{\partial \phi}{\partial r}
\]

integration yields

\[
\boxed{
\phi = \frac{Q}{2\pi}\ln r
}
\]

\subsection{Stream Function}
\label{subsec:source-and-sink-flow-stream-function}

The stream function is

\[
\boxed{
\psi
=
\frac{Q}{2\pi}\theta
}
\]
