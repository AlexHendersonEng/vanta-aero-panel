\section{Introduction}
\label{sec:introduction}

Solutions of Laplace's equation can be superposed to construct more
complicated flow fields.
Several fundamental solutions, known as \emph{elementary flows}, form
the building blocks of classical potential flow theory.

The most important elementary flows are:

\begin{itemize}
    \item Uniform flow
    \item Source and sink flow
    \item Doublet flow
    \item Free vortex flow
\end{itemize}

Each flow possesses a velocity potential $\phi$ and a stream function
$\psi$, both satisfying Laplace's equation.
