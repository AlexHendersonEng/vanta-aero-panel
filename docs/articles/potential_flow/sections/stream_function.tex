\section{Stream Function}
\label{sec:stream-function}

For incompressible two-dimensional flow,
\[
u=\frac{\partial \psi}{\partial y},
\qquad
v=-\frac{\partial \psi}{\partial x}
\]

Using the irrotationality condition yields
\[
\nabla^2 \psi =0
\]

Thus, the stream function also satisfies Laplace's equation.
